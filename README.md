## Fullstack Inventory Management Dashboard

An Inventory Mangagement Dashboard app which is fully responsive and has theme toggle feature. It is deployed using Vercel.
It uses Next.js for the frontend, styled with Tailwind CSS featuring Material UI Data Grid for complex data handling. The state management is streamlined with Redux Toolkit, supplemented by Redux Toolkit Query for data fetching.
The backend is powered by Node.js using Prisma as the ORM to facilitate database interactions.

## Tech Stack
- Next JS
- Tailwind
- Redux Toolkit
- Redux Toolkit Query
- Material UI Grid
- Node.js
- Prisma
- Postgres
